O — Optimisation B1: Precomputed Anchor Embeddings

KNN classification uses 146 anchor phrases that were previously encoded on-the-fly at every startup. Running precompute_anchors.py once and caching to .npy files gives a 392x speedup per file and saves ~239 ms total at startup.

O — Optimisation B2: Faster-Whisper INT8 vs Standard Whisper FP32

INT8's key advantage is RAM, not speed — quantisation halves the model weight size, saving 75 MB in a system where RAM is shared across 4 concurrent threads. Speed is secondary since ASR runs async and never blocks inference.
